# 🚀 Инструкция по установке и запуску АнтиБанк

## Требования

### Обязательные компоненты:
- **Node.js 18+** 
- **Android Studio** с установленным Android SDK
- **JDK 11** или выше
- **Физическое Android устройство** или эмулятор

### Переменные окружения:
```bash
# Добавьте в ~/.bashrc или ~/.zshrc (Linux/Mac) или переменные среды Windows:
export ANDROID_HOME=$HOME/Library/Android/sdk  # Mac
export ANDROID_HOME=%LOCALAPPDATA%\Android\Sdk  # Windows
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools
```

## 📦 Установка

### 1. Клонирование проекта
```bash
git clone <repository-url>
cd antibank
```

### 2. Установка зависимостей
```bash
npm install
# или
yarn install
```

### 3. Предварительная настройка Android
```bash
# Убедитесь, что Android SDK установлен
npx expo doctor

# Если нужно, установите недостающие компоненты:
npx expo install --fix
```

## 🔧 Настройка разрешений

### Автоматическая настройка (рекомендуется):
1. Запустите приложение
2. Перейдите на вкладку "Разрешения"
3. Нажмите "Запросить все разрешения"
4. Следуйте инструкциям на экране

### Ручная настройка:
Если автоматическая настройка не работает:

1. **Доступ к контактам:**
   - Настройки → Приложения → АнтиБанк → Разрешения → Контакты → Разрешить

2. **Управление звонками:**
   - Настройки → Приложения → АнтиБанк → Разрешения → Телефон → Разрешить все

3. **Отображение поверх приложений:**
   - Настройки → Приложения → АнтиБанк → Дополнительно → Отображение поверх других приложений → Разрешить

4. **Приложение по умолчанию для звонков (Android 10+):**
   - Настройки → Приложения → Приложения по умолчанию → Приложение для звонков → АнтиБанк

## 🚀 Запуск приложения

### Режим разработки:

#### Вариант 1: Expo Development Build
```bash
npx expo start --android
```

#### Вариант 2: Прямой запуск на устройстве
```bash
npx expo run:android
```

### Продакшн сборка:

#### Локальная сборка APK:
```bash
# Сборка debug APK
cd android
./gradlew assembleDebug

# APK будет в: android/app/build/outputs/apk/debug/app-debug.apk
```

#### Сборка через EAS (рекомендуется):
```bash
# Установка EAS CLI
npm install -g @expo/cli

# Настройка проекта
npx eas build:configure

# Сборка для Android
npx eas build --platform android
```

## 📱 Подключение устройства

### Физическое устройство:
1. Включите "Режим разработчика" на Android
2. Включите "Отладка по USB"
3. Подключите устройство к компьютеру
4. Разрешите отладку при появлении запроса

### Эмулятор:
1. Откройте Android Studio
2. AVD Manager → Create Virtual Device
3. Выберите устройство с API 24+ (Android 7.0+)
4. Запустите эмулятор

## 🔍 Проверка работы

### Проверьте установку:
```bash
# Проверка подключенных устройств
adb devices

# Должно показать ваше устройство/эмулятор
```

### После установки приложения:
1. **Откройте АнтиБанк**
2. **Предоставьте все разрешения** (критически важно!)
3. **Активируйте блокировщик** на главном экране
4. **Проверьте статус** - должен быть зеленый "Защита активна"

## 🚨 Решение проблем

### Приложение не блокирует звонки:
```bash
# Проверьте разрешения через adb:
adb shell dumpsys package com.antibank.callblocker | grep permission
```

### Проблемы с сборкой:
```bash
# Очистка кэша
npx expo start --clear

# Очистка Gradle (Android)
cd android && ./gradlew clean

# Переустановка зависимостей
rm -rf node_modules && npm install
```

### Проблемы с автозапуском:
1. **MIUI (Xiaomi):** Настройки → Автозапуск → АнтиБанк → Включить
2. **EMUI (Huawei):** Настройки → Приложения → АнтиБанк → Батарея → Автозапуск
3. **ColorOS (Oppo):** Настройки → Батарея → Оптимизация энергопотребления → АнтиБанк → Не оптимизировать

### Сервис не работает в фоне:
1. Отключите оптимизацию батареи для АнтиБанк
2. Добавьте приложение в автозагрузку
3. Разрешите работу в фоне

## 📞 Тестирование

### Безопасное тестирование:
1. Добавьте тестовый номер в черный список
2. Попросите друга позвонить с этого номера
3. Звонок должен быть заблокирован
4. Проверьте историю в приложении

### Тестирование микровызовов:
1. Включите блокировку микровызовов
2. Попросите сделать очень короткий звонок (1-2 секунды)
3. Звонок должен появиться в истории как "Микровызов"

## 🔐 Важные замечания

- **Работает только на Android** (iOS не поддерживает API блокировки)
- **Требует Android 7.0+** для базовой функциональности
- **Android 10+** рекомендуется для полной функциональности
- **Некоторые прошивки** могут требовать дополнительных настроек
- **Данные хранятся локально** - никакой передачи в интернет
